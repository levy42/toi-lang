Functions

Declaration

fn add(a, b)
  return a + b

Default Parameters

fn add(a, b = 10)
  return a + b

Variadic Parameters

fn sum(*vals)
  ...

Type Hints

Type hints are supported in parameter lists:

fn f(user_id: int, name: str)
  ...

Type hints are metadata (inspectable), not a full static type system.

Anonymous Functions

fn(x)
  return x * 2

Decorators

Decorator syntax is supported for function declarations:

@decorator
fn f()
  ...

Named and Spread Call Arguments

  • Named args: f(a=1, b=2)
  • Spread args: f(*arr)

Note: named call arguments use =. Expression assignment uses := (walrus), for example: x := expr.

Multiple Return Values

fn pair()
  return 1, 2
