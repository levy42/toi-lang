# Standard Library

Native modules are loaded with `import`:

```pua
math = import math
```

Available native modules:

- `math`
- `time`
- `io`
- `os`
- `coroutine`
- `string`
- `table`
- `socket`
- `thread`
- `json`
- `template`
- `http`
- `inspect`
- `binary`
- `struct`
- `btree`
- `uuid`

Global core functions are documented in `docs/stdlib/core.md`.

Pua-implemented helper libraries are documented in `docs/stdlib/pua-libraries.md`.
